# Grace E. Schools Website

A professional, colorful website for Grace E. Schools - a premier educational institution in Lagos, Nigeria.

## About Grace E. Schools

Grace E. Schools is a well-established private educational complex located in Gbagada Estate, Phase I, Lagos, Nigeria. Founded in 1968 by Deaconess Mrs. Grace Bisola Oshinowo, the institution provides comprehensive education from creche through secondary education.

## Website Structure

The website consists of 6 main pages:

### 1. Home Page (index.html)
- Hero section with call-to-action buttons
- Introduction to Grace E. Schools
- Overview of the three educational levels (Creche, Primary, College)
- Core values section
- Statistics and achievements

### 2. About Us (about.html)
- School history and legacy
- Mission, vision, and philosophy
- Why choose Grace E. Schools
- Detailed core values
- Educational approach

### 3. Creche (creche.html)
- Information about infant care (6 weeks - 2 years)
- Safety and hygiene standards
- Daily care routines
- Age-appropriate activities
- Professional caregivers

### 4. Primary School (primary.html)
- Grace Children School (Nursery - Grade 6)
- Comprehensive curriculum (Nigerian/British blend)
- Grade level breakdown
- Modern facilities
- Teaching approach

### 5. College/High School (college.html)
- Grace High School (JSS 1 - SS 3)
- Academic tracks: Science, Arts, Commercial
- Day and boarding options
- Co-curricular activities
- Student support services

### 6. Contact (contact.html)
- Contact information
- Interactive contact form
- Admission process overview
- FAQs
- Location details

## Design Features

- **Colorful & Professional**: Modern gradient backgrounds, vibrant color scheme
- **Responsive Design**: Mobile-friendly layout that adapts to all screen sizes
- **Consistent Navigation**: Easy-to-use navigation across all pages
- **Visual Hierarchy**: Clear sections with icons and visual elements
- **Accessibility**: Semantic HTML and readable font sizes
- **Interactive Elements**: Hover effects, animations, and transitions
